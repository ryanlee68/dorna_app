Startup Programs

Configure the list of programs that run automatically when the robot starts up. Use the list below to add or remove programs.

Notice
  • To add a program to the list, simply insert a new line in the list with the program's complete path specified. For example,
    • Python program: /full/path/to/the/python_file.py
    • Blockly program: /full/path/to/the/blockly_file.blk
  • Programs will be executed during the startup, all at the same time.
  • Use one line for each program.
  • Use # (hashtag symbol) at the beginning of each line to comment that line.
  • Empty lines will be ignored.
