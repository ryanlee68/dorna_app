Startup Programs

The programs that run automatically when the robot starts up. Use the list below to add or remove programs from the list.

Notice
  • Use one line for each program.
  • Programs will be executed during the startup, all at the same time.
  • Use # (hashtag symbol) at the beginning of each line to comment that line.
  • Empty lines will be ignored.
  • To include a Python program found at full/path/to/the/python_file.py, add a line to the list that says python3 full/path/to/the/python_file.py.
  • To include a Blockly program found at full/path/to/the/blockly_file.blk, add a line to the list that says python3 full/path/to/the/blockly_file.py (change the .blk extension to .py).
